# 🏓 乒乓球 AI 教学系统

基于 MediaPipe Pose 的智能乒乓球动作分析教学系统。

## 功能

- 🎯 **仪表盘** — 训练数据总览与进度追踪
- 📊 **训练记录** — 历史训练数据分析
- 🤖 **动作分析** — AI 实时骨骼检测与分析
- 📈 **成长复盘** — 技能提升可视化
- 📅 **课程计划** — 个性化训练计划管理
- 🔄 **动作对比** — 你的动作 vs 马龙标准动作

## 使用

打开页面后，进入「动作对比」模块，点击「开启摄像头」即可使用。

> 摄像头功能需要 HTTPS 环境（GitHub Pages 自动提供）。
>
> 推荐使用 Chrome 或 Edge 浏览器以获得最佳体验。

## 技术栈

- Chart.js (图表)
- Font Awesome (图标)
- Google Fonts (字体)
- MediaPipe Pose (骨骼识别)
- localStorage (本地存储)

## 动作库

包含马龙 12 项标准技术动作数据：
正手攻球、反手攻球、正手下旋拉、反手下旋拉、正手搓球、反手搓球、正手挑打、反手挑打、反手拨球、发球、前后步法、左右步法
